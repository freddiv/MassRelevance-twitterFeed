Massrelevance Twitter Feeds

Code Challenge: implement the widget, which will render N last tweets from the given JSON feed into specified HTML container. Widget should update the list with the specified interval, by adding new tweets and removing an equal number of tweets so the widget is always displaying N tweets. Just new elements should be rendered; existed should not. Display time, author and message text for every list item.

Kindle Feed

Soccer Feed

Details

Arguments to accept:

  1. listEl: the element where the feed will be displayed
  2. lastID: the id of the most recent rendered tweet-this defaults to "" and is updated whenever the feed is updated. Pass this to the api to return only the tweets newer than this tweet
  3. limit: Amount on list items to render - defaults to 20
  4. feed: the JSON feed URL - defaults to the kindle feed.
  5. interval: The interval to update feed (* 1000)
